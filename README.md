# Desk Pet V2
The Desk Pet (V2) is a gadget you place on your desk to display useful information, centered around the theme of Fallout
it displays the time, room conditions like temperature and humidity and provides useful functions like a stopwatch, timer and is upgradeable for IoT gadget control.

This project is created by Santiago Strajijar. This is designed as a weekend project for anyone to complete. The Desk Pet requires 3D printed parts and a custom PCB which can either be ordered online or manually created using milling or etching.
